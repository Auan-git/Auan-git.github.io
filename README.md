# 守住你的"不服输"（keep your identity small）

> 书呆子之所以不受欢迎，并不是因为他们沉迷于思考，而是因为他们没有时间去学习社交技能，他们把时间花在了对真实世界的好奇心上。

一个计算机专业学生的学习博客，记录从 Hello World 到复杂系统的思考轨迹。

## 博客理念

「像画家一样思考，像黑客一样创造」

- **画家思维**：让技术实践保有审美，写代码不是堆砌字符，而是像调配色彩一样平衡逻辑与可读性
- **黑客精神**：拆解规则、重构可能性，敬畏语法的底线，再追问「还能怎么做」


## 技术栈

| 组件     | 技术                                                         |
| -------- | ------------------------------------------------------------ |
| 静态框架 | [Hexo](https://hexo.io/) v8.1.1                              |
| 主题     | [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) |
| 部署     | GitHub Pages                                                 |
| 域名     | [xingqiwu.net.cn](https://xingqiwu.net.cn)                   |

## 本地开发

```bash
# 安装依赖
npm install

# 启动本地服务器
npm run server

# 构建静态文件
npm run build

# 部署到 GitHub Pages
npm run deploy
```

## 项目结构

```
source/
├── _posts/          # 博客文章

├── about/           # 关于页面
├── archives/        # 归档页面
├── categories/      # 分类页面
└── tags/            # 标签页面
```
