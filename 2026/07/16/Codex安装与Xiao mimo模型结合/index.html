Codex安装与Xiao mimo模型结合

本教程实现：本地 mimo2codex 代理 → 将 Codex 请求转发至小米 MiMo 大模型，全程适配 Windows 环境，规避编译、版本、密钥、联网搜索等常见报错。

安装Codex

由于codex是openAI的产品 所以安装时需要科学上网
通过官网：https://openai.com/codex/for-work/ 下载Codex微软软件安装包（Codex Installer.exe）
运行它。安装好就关闭就好了

安装Node.js和npm

作用：在任何地方运行 JavaScript
Node.js® 是一个免费、开源、跨平台的 JavaScript 运行时环境，它让开发人员能够创建服务器、 Web 应用、命令行工具和脚本。

链接：https://nodejs.org/

不要安装最新版本：要安装较老版本：下载安装：https://nodejs.org/dist/v20.18.0/

安装后验证：
