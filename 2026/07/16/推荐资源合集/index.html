推荐资源合集

推荐资源合集

给 CS 新生的精选资源清单

每个类别只推荐最好的，不贪多。资源太多反而不知道看什么——先把这些用好，以后按需扩展。

视频课程

最高效的学习方式之一。但要跟着敲代码，不要只看。

资源 说明 链接 费用
CS50（哈佛大学） 全世界最好的计算机入门课，没有之一。从零讲起，节奏好、质量高、老师讲课极有感染力。强烈建议军训期间开始看第一课。 B站搜”CS50”，有中文字幕 免费
翁恺 C 语言（浙大） 国内最好的 C 语言入门课。学校教 C 的时候配合着看，效果翻倍。 B站搜”翁恺 C 语言” 免费
3Blue1Brown 神经网络系列 用动画讲数学和神经网络，神作。看完你会对 AI 底层原理有直觉性的理解。 B站搜”3Blue1Brown 神经网络” 免费
3Blue1Brown 线性代数系列 用动画讲线性代数的本质，比课堂上老师讲的直观一万倍。大一学线代之前看一遍，开挂。 B站搜”3Blue1Brown 线性代数” 免费
李宏毅机器学习课 台湾教授，中文授课，讲得非常好懂。想了解机器学习入门的首选。 B站搜”李宏毅 机器学习” 免费
MIT 6.006 算法导论 MIT 的算法课，B站有中文字幕。大一下学数据结构/算法的时候配合看。 B站搜”MIT 6.006” 免费
CS144 计算机网络（斯坦福） 斯坦福的网络课，大二学计算机网络的时候配合看。 B站搜”CS144” 免费
MIT 6.S081 操作系统（MIT） MIT 的操作系统实验课，大二学 OS 的时候配合看。做一个真正的操作系统内核。 B站搜”MIT 6.S081” 免费
吴恩达机器学习课程 Coursera 上的经典课程，英文但有中文字幕。比李宏毅的更偏理论。 coursera.org 免费旁听

书籍

不需要全买。每阶段挑一本，读完再换下一本。

入门科普

书名 说明
《编码：隐匿在计算机软硬件背后的语言》 不是教你写代码的，而是用最通俗的方式告诉你计算机是怎么工作的。从电灯开关讲到 CPU，非常好读，适合大一。
《计算机科学导论》 如果你想要一本全面的 CS 概览书，这本可以翻翻。不需要通读，当参考书用。

编程语言

书名 说明
《C Primer Plus》 C 语言最经典的教材，厚但讲得清楚。配合学校课程使用。
《C和指针》 理解 C 语言指针的神器。《C Primer Plus》读完后可以看这本深入。
《Python 编程：从入门到实践》 Python 入门首选，前半部分讲语法，后半部分带你做三个真实项目。
《流畅的 Python》 Python 进阶书。大一先不碰，等你写了半年 Python 再看，会让你功力大增。
《Effective Java》 Java 进阶经典。等学完 Java 基础再看。
《C++ Primer》 C++ 最权威的教材。比《C++ Primer Plus》质量高，适合有 C 基础后阅读。

数据结构与算法

书名 说明
《大话数据结构》 数据结构入门，用大白话和图解讲，比课本好懂很多。适合大一下配合课程使用。
《算法图解》 算法入门，用大量图解讲常见算法，非常友好。适合零基础。
《算法导论》（CLRS） 算法领域的”圣经”，非常硬核。大一下可以先翻一翻，大二大三再深入读。
《剑指 Offer》 面试算法题精选。大二大三准备面试的时候刷。

系统层

书名 说明
《深入理解计算机系统》（CSAPP） CS 本科生必读，没有之一。从程序员视角理解计算机系统，打通从硬件到软件的完整认知。大二开始读，每章都值得反复看。
《Operating Systems: Three Easy Pieces》（OSTEP） 操作系统最好的教材之一，免费在线阅读（ostep.org）。比国内教材好懂很多。
《计算机组成原理》（唐朔飞） 国内教材，配合学校课程使用。
《计算机网络：自顶向下方法》 计算机网络经典教材，从应用层讲起，比从物理层讲起的教材好懂很多。
《数据库系统概论》（王珊） 国内数据库经典教材。
《编译原理》（龙书） 编译原理的”圣经”。很厚很硬核，大三再看。

网络与安全

书名 说明
《图解 HTTP》 用图解的方式讲 HTTP 协议，非常易读。大一就能看。
《图解 TCP/IP》 用图解的方式讲 TCP/IP 协议栈。配合计算机网络课程使用。
《白帽子讲 Web 安全》 Web 安全入门，国内作者，中文原生。对安全方向感兴趣可以翻翻。

AI 与机器学习

书名 说明
《动手学深度学习》（d2l.ai） 李沐写的深度学习教材，配合代码，免费在线阅读。学完线代和概率后可以开始看。
《机器学习》（西瓜书） 周志华的经典教材，国内 ML 方向必读。偏理论，需要数学基础。
《统计学习方法》（李航） 另一本 ML 经典教材，比西瓜书更偏数学推导。
《Deep Learning》（花书） 深度学习的”圣经”，AI 方向必读。大三再看。

学习方法与职业发展

资源 说明
《提问的智慧》（How To Ask Questions The Smart Way） 不是一本书，是一篇文章。教你怎么正确提问。网上搜中文翻译版，读一遍只需要 20 分钟，受益四年。
《自学是门手艺》（李笑来） 讲自学方法论，免费在线阅读。适合大一读，建立正确的学习观。
CS 自学指南（csdiy.wiki） 北大同学整理的计算机自学资源合集，非常全面。按方向分类，有推荐路线。

平台 说明 链接 适合阶段

洛谷 国内算法竞赛平台，入门题目比 LeetCode 更友好，适合零基础。 luogu.com.cn 大一就可以
蓝桥杯练习系统 蓝桥杯竞赛的官方练习平台，难度适中。 lx.lanqiao.cn 大一就可以
AcWing 国内算法竞赛平台，有配套视频课程，适合系统学习算法。 acwing.com 大一下开始
Codeforces 国际算法竞赛平台，题目质量高，适合有一定基础后挑战。 codeforces.com 大二开始
Kaggle 数据科学和机器学习竞赛平台。做 ML 项目、拿竞赛经历的最佳去处。 kaggle.com 大二大三

数据库工具

工具 用途 链接 说明
MySQL 关系型数据库 mysql.com 最流行的开源数据库之一。大二学数据库课程会用到。
PostgreSQL 关系型数据库 postgresql.org 功能最强大的开源数据库。比 MySQL 更适合复杂查询。

Navicat 数据库管理工具 navicat.com 功能强大的数据库可视化工具。付费，但非常方便。

容器与虚拟化

工具 用途 链接 说明
VMware Workstation Pro 虚拟机软件 vmware.com 在 Windows 上运行完整的 Linux/其他操作系统。2024 年起个人使用已免费。学 Linux 必备。
VirtualBox 虚拟机软件 virtualbox.org Oracle 出品的免费开源虚拟机。功能不如 VMware 但完全免费。
WSL2 Windows 子系统 for Linux 微软商店搜”Ubuntu” 在 Windows 上直接运行 Linux，不需要虚拟机。性能好、启动快。Windows 用户首选。
Docker 容器化工具 docker.com 轻量级”虚拟化”，把应用和环境打包在一起。大二大三会用到。
Proxmox VE 服务器虚拟化平台 proxmox.com 在一台物理机上运行多个虚拟机。有闲置电脑的可以折腾。

WSL vs VMware vs Docker，怎么选？

场景 推荐
日常使用 Linux 命令行、跑简单的 Linux 程序 WSL2（最轻量、最方便）
需要完整的 Linux 图形桌面、学内核/驱动 VMware + Ubuntu 虚拟机（最完整）
部署项目、复现环境、容器化开发 Docker（最专业）
大一新生，只想先用起来 先装 WSL2，有需要再装 VMware

终端与命令行工具

工具 用途 链接 说明
Windows Terminal Windows 终端 微软商店搜”Windows Terminal” 微软官方终端，比 CMD 和 PowerShell 好看好用。Windows 必装。
Oh My Zsh 终端美化与增强 ohmyz.sh 让终端更好看、补全更智能。Linux/Mac 标配。
tmux 终端多路复用 github.com/tmux/tmux 一个终端窗口分成多个面板。连接服务器时尤其有用，断开后程序不会停。

网络工具

工具 用途 链接 说明
Wireshark 网络抓包分析 wireshark.org 捕获并分析网络数据包。学计算机网络时的神器，亲眼看 TCP 三次握手。强烈推荐。
Postman API 测试工具 postman.com 测试后端接口（API）的必备工具。发送 HTTP 请求、查看响应。
curl 命令行 HTTP 工具 自带 在终端里发 HTTP 请求。调试 API、测试连通性。
nmap 网络扫描工具 nmap.org 扫描网络中的设备和开放端口。学安全方向会用到。
FileZilla FTP 文件传输工具 filezilla-project.org 通过 FTP/SFTP 传输文件到服务器。免费开源。
MobaXterm SSH 远程连接工具 mobaxterm.mobatek.net Windows 上最好用的 SSH 客户端，集成文件传输、X11 转发等。免费版够用。
FinalShell SSH 远程连接工具 hostbuf.com 国产 SSH 工具，界面友好，支持文件传输和服务器监控。

SSH 连接工具怎么选？

  • Windows 用户： MobaXterm（最全能）或 FinalShell（国产友好）

  • Mac/Linux 用户： 直接用终端 ssh 命令就行

  • VS Code 用户： 安装 Remote - SSH 扩展，直接在 VS Code 里连服务器写代码（最推荐）
